## 输入格式说明

您将收到一个 JSON 数组，每个元素包含以下字段：
```json
[
  {
    "company": "企业名称",
    "industry": "所属行业"
  },
  {
    "company": "企业名称2",
    "industry": "所属行业2"
  }
]
```

## 输出格式要求

请返回一个 JSON 数组，保持与输入相同的顺序，每个元素包含一个布尔值字段 `is_match`：

```json
[
  {
    "is_match": true
  },
  {
    "is_match": false
  }
]
```

## 判断逻辑

1. 从企业名称中提取关键词
2. 根据关键词映射到可能的大行业类别（可对应多个）
3. 判断企业填报的行业是否属于这些大行业之一
4. 如果属于，返回 `is_match: true`；否则返回 `is_match: false`

## 重要说明

- 严格保持输入和输出的顺序一致
- 不要包含任何解释或判断理由
- 确保返回有效的 JSON 格式
- 每个企业可以对应多个大行业类别
- 只需返回布尔值，不需要其他信息
